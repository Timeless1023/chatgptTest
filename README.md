# 5分钟生存战（类吸血鬼幸存者）

一个纯前端 Canvas 小游戏：5 分钟内不死亡即获胜。

## 试玩方式

```bash
python3 -m http.server 4173
```

然后浏览器打开：

- `http://127.0.0.1:4173`

## 操作

- 电脑：`WASD` 移动
- 手机：左下角虚拟摇杆移动
- 角色会自动进行远程攻击

## 规则

- 敌人会持续生成并追踪玩家，且会随时间逐步变强
- 2:30、4:30 会出现 Boss
- 敌人死亡掉经验球，靠近自动吸收
- 升级时会出现三选一强化（含多重弹道、连锁雷击、爆燃爆炸等）
- 玩家碰到任意敌人会立刻死亡
- 倒计时 5:00 结束前存活即胜利
